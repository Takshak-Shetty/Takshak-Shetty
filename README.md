## 👋 About Me

<img align="right" width="240" src="https://raw.githubusercontent.com/Anmol-Baranwal/Cool-GIFs-For-GitHub/main/rocket_flying_gif.gif"/>

I'm **Takshak Shetty**, a Cybersecurity Engineering student building things at the intersection of **secure systems** and **practical software**. Currently an **IT Intern at Bajaj Finserv**, working on real-world backend problems, and leading technical culture on campus as **President of the CSI Student Branch**.

I care about writing code that's clean *and* defensible — I like knowing not just how a system works, but how it breaks.

- 🔭 **Working On:** Real-world backend systems @ Bajaj Finserv
- 🌱 **Learning:** Cloud security, scalable architecture, AWS & Azure
- 🎯 **Goal:** Software/Backend Engineer with a security-first mindset
- 🏛️ **Leadership:** President, CSI Student Branch — hackathons, tech events, community
- 💡 **Fun Fact:** I enjoy breaking systems as much as I enjoy building them
